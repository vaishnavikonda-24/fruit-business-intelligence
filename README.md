# 🍎 Fruit KPI Dashboard Analysis

## 📖 Project Overview

This project focuses on **KPI Analysis and Dashboard Development** using a Fruit Classification Dataset. The objective is to analyze fruit data, calculate key performance indicators (KPIs), create meaningful visualizations, and generate business insights through an interactive dashboard.

The project demonstrates the practical application of **Data Analytics** and **Business Intelligence** techniques using Python.

---

## 🎯 Objectives

- Analyze the fruit dataset.
- Calculate important KPIs.
- Create data visualizations.
- Develop a dashboard for reporting.
- Generate business insights from the data.

---

## 📂 Dataset Information

The dataset contains the following attributes:

- Weight
- Colour
- Label (Apple/Banana)
- Fruit_Size
- Label_Code

### Dataset Summary

- Total Records: 17
- Fruit Categories: Apple and Banana
- Total Features: 5

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Value |
|------|------|
| Total Fruits | 17 |
| Apple Count | 8 |
| Banana Count | 9 |
| Average Weight | 334.06 g |
| Maximum Weight | 382 g |
| Minimum Weight | 277 g |

---

## 📈 Data Visualizations

The following visualizations were created:

### 1. Fruit Distribution (Pie Chart)
Shows the percentage distribution of Apples and Bananas.

### 2. Fruit Size Distribution (Bar Chart)
Displays the count of fruits based on size categories.

### 3. Weight Distribution (Histogram)
Represents the frequency distribution of fruit weights.

### 4. Average Weight by Fruit Type (Bar Chart)
Compares the average weight of Apples and Bananas.

---

## 💡 Business Insights

- Bananas slightly outnumber Apples in the dataset.
- The average fruit weight is 334.06 grams.
- Weight ranges between 277 g and 382 g.
- Fruit size and weight are important classification features.
- Dashboard visualizations improve data interpretation and decision-making.
